Currently only Xposed-based activation is supported, and the activation steps are as follows:

1. Confirm that the device is rooted and the Xposed framework is installed. It is recommended to use [Lsposed framework](https://lsspoke.org)
2. Download and install ShortX App
3. Open Xposed manager, select ShortX, select Android system and system UI
4. Restart the device
